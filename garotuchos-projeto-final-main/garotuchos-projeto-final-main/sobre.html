Proposta

A proposta primordial do trabalho é desenvolver um portal referente a cidade de Guarulhos, abordando pontos turísticos, históricos, hospitalares, de lazer em geral e tudo que cerca a grande Guarulhos. Para tal ato, serão reunidas diversas informações a respeito de Guarulhos, e estarão disponibilizadas no site para fácil acesso, site esse que contará com uma estrutura a base de HTML, CSS e JavaScript. Como resultado, buscamos facilitar a busca de informações em volta da cidade, sendo útil tanto para moradores, quanto turistas. Por fim, ressalta-se que o projeto contribuirá para qualquer um, de maneira gratuita e fácil manuseio por parte do cliente.
